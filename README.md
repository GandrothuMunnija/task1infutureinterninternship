
# Business Sales Performance Analysis Dashboard

## Project Overview
This project presents an interactive Business Sales Performance Analysis Dashboard created using Microsoft Excel. The dashboard helps analyze sales performance, profitability, products, and regional trends using visual reports and interactive filters.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

---

## Dataset
Superstore Sales Dataset

---

## Key Insights
- Identified the highest sales month
- Determined the best-selling product
- Analyzed the best-performing region
- Found the most profitable category

---

## Dashboard Features
- Interactive slicers for dynamic filtering
- Monthly sales trend analysis
- Regional performance comparison
- Category-wise profitability analysis
- Product performance visualization

---

## Conclusion
The dashboard provides meaningful business insights that help in understanding sales trends, customer preferences, and profitability. It enables better decision-making through interactive data visualization and analysis.
